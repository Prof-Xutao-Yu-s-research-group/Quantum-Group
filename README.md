## 量子计算研究小组介绍

量子计算研究小组依托于东南大学**毫米波全国重点实验室**和江苏省**紫金山实验室**，现主要研究主要分四个方向，分别是**量子计算机的刻画和可靠性基准验证体系**、**量子线路智能设计与优化**、**Quantum Computation for AI/AI for Quantum Computation**、**量子计算在通信系统和其它领域中的应用**，团队现有**博士研究生7名**，**硕士研究生6名**；研究小组内部**氛围轻松融洽**，交流自由，**科研节奏有序**，鼓励探索和尝试；**项目稳定**，**经费充足**，每年进行多次**团建活动**，同时非常支持参加**国内外学术会议**，鼓励同学积极拓宽视野，交流成果；欢迎对量子研究感兴趣的同学加入我们量子计算研究小组攻读**硕士/博士学位**（邮箱：yuxutao@seu.edu.cn，可附带简历）。
